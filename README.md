EcoFinds 🌿
EcoFinds ek sustainable marketplace application hai jo users ko eco-friendly products kharidne, bechne aur manage karne mein madad karti hai. Yeh app Flutter ka upyog karke banayi gayi hai aur Material Design 3 ke modern UI principles ko follow karti hai.

Features ✨
Product Discovery: Ek centralized product feed jahan users naye items browse kar sakte hain.

Listing Management: Users apne khud ke products list kar sakte hain aur unhe manage kar sakte hain.

Shopping Cart & Purchases: Items ko cart mein add karne aur unki complete history track karne ki suvidha.

Order Tracking: Alag-alag roles ke liye dedicated screens jaise 'My Orders' (customers ke liye) aur 'Order Management' (sellers ke liye).

State Management: Provider package ka upyog karke smooth aur efficient data handling.

Authentication: Secure login aur signup flow jo user ke data ko isolate rakhta hai.

Tech Stack 🛠️
Framework: Flutter (SDK ^3.8.1)

Language: Dart

State Management: Provider

Local Storage: Shared Preferences (User preferences save karne ke liye)

Image Handling: Image Picker

HTTP: Http package (Future API integration ke liye)

Project Structure 📁
App ka logic aur UI in main directories mein divided hai:

lib/models/: Data models define karta hai (Cart, Order, Product, User, etc.).

lib/providers/: App ka business logic aur state management handle karta hai.

lib/screens/: Saare UI screens (Login, Dashboard, Navigation, etc.).

lib/main.dart: App ka entry point aur global theme configuration.

Getting Started 🚀
Is project ko local machine par setup karne ke liye:

Repository Clone Karein:

Bash
git clone [repository-url]
Dependencies Install Karein:

Bash
flutter pub get
App Run Karein:

Bash
flutter run
